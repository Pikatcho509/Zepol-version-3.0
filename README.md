# Zepòl Web Version

This is the web version of the Zepòl app, a social platform for sharing in Haitian Creole.

## Features

- View posts from users
- Like posts
- Add new posts

## How to Run

1. Ensure you have Python installed (comes with most systems).
2. Open a terminal in the `web` folder.
3. Run: `python -m http.server 8000`
4. Open your browser and go to `http://localhost:8000`

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Notes

This is a simple static version. In a real app, you'd connect to a backend for data persistence.
